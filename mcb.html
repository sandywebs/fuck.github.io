Blind Coding

Rules :

1. Each participant has to write the code for the problem statement, with MONITOR OFF

2. Participants whose code gets executed will be given higher preference.

3. The second preference will be given to code with the least number of errors

4. In case of tie in number of errors then timing will be considered

5. Shortlisted participants of Round I will appear for Round II for which above rules will be applicable
